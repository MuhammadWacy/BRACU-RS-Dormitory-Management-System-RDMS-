# BRACU-RS-Dormitory-Management-System-RDMS-
This is our database project from CSE370 course of BRAC University where I am trying to solve a practical problem faced in the residential semesters(RS) of  my university where students have to stay in their alloted dormitories. There is a lack of a dormitory management system which would have made the RS of students and authorities a lot easier. The goal is to take inspiration from the inconveniences we faced in RS and solve it practically using the database concepts learned throughout the course. We hope that some form of the optimized version of the project or  at least its fundamental ideas would be adapted by the university authority to make the RS life more convenient for everyone. The project uses html, css in the frontend and php, mysql in the backend.
